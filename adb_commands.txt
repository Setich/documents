Список подключенных устройств - adb devices
Установить приложение - adb install имя_приложения.apk
Android консоль - adb shell
Список устаноленных приложений - adb shell pm list packages(в конце добавить -s - олько системные приложения, -3 — сторонние, -f покажет пути установки пакетов,-d — отключенные приложения)
Найти package определенного приложения - adb shell pm list packages | grep имя_приложения
Отключить приложение - pm disable com.android.имя_приложения
Очистить данные приложения - pm clear com.android.имя_приложения
Удалить приложение - pm uninstall com.android.имя_приложения
Запись лога с устройства adb logcat
Запись лога нужной программы adb logcat | grep -rnw "com.android.имя_программы" > путь_куда_записать_лог/название_файла.log
Остановить запись лога - -С
Сделать скриншот - adb shell screencap путь_куда_зальётся_скриншот/имя_скриншота.png
Отправить файл на телефон - adp push путь_от_куда/имя_файла путь_куда/имя_файла
Отправить файл с телефона на комп - adb pull путь_от_куда/имя_файла путь_куда/имя_файла
